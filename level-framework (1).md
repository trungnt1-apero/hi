# English Learning App — Level Framework

> **Standard**: CEFR | **Last updated**: 2026-05

## Overview

| Level | CEFR | Label | Target learner |
|---|---|---|---|
| Elementary | A1–A2 | Beginner | Young learners, first contact with English |
| B1 | B1 | Intermediate | Can handle everyday situations, familiar topics |
| B2 | B2 | Upper-Intermediate | Fluent with native speakers, complex texts |
| C1 | C1 | Advanced | Academic / professional fluency, implicit meaning |

---

## Notes for product team

| Topic | Detail |
|---|---|
| **Placement test** | Learners should be placed via an in-app adaptive test covering all four skills before their first session |
| **Level progression** | Advancement requires demonstrating competency in at least 80% of Can-Do outcomes across all skill areas |
| **Content mapping** | Each unit should be tagged with the specific Can-Do outcome(s) it addresses to enable progress tracking |
| **C2 scope** | C2 (Mastery) is not in current scope — add when roadmap supports native-speaker-equivalent content |
